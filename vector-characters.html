Orange

Vector Character System

Orange Cameroon’s youth Orange Pulse needed illustrations unique to the platform, that would be used to communicate exclusively on Pulse. The idea was to create a universe perculiar to Orange Pulse

All the illustrations' elements were grouped into categories and a smart library of reusable components was built. Easy to use, by any designer/illustrator on the team, and ship creatives in record time.
